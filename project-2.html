Assistive Autonomous Robot

The objective was to design and construct an autonomous robot that replicates a real-world application: an assistive robot in Germany that enters a kitchen, turns off a stove, finds a plate, and returns it to an elderly person. For this project, the stove was represented by a 1 kHz red beacon, and the plate by a 10 kHz green beacon.

Project Overview

The robot needed to 1. Find the red beacon and press its button. 2. Locate the green beacon, secure it, and remove it from the arena. 3. Return to the starting position autonomously avoiding obstacles on the way.

I contributed to this project with the build of the robot and soldering of the IR board. Constructing a lightweight, stable robot using VEX parts along with optimizing the button presser and beacon catcher for reliability. Assembling the IR board, ensuring proper beacon detection.
